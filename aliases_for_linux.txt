  Для создания псевдонимов (aliases) для Linux надо открыть файл ~/.bashrc и записать в него в конце команду следующего вида:

	alias gi='git init'

где gi - наш будущий псевдоним (alias), git init - команда, которую будет выполнять данный alias 

  Также можно целую последовательность команд заключить в один alias, для этого между командами ставим && ,например:

	alias gac='git add && git commit'
