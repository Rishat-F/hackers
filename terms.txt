git add - трекинг файла, будет следить за его изменениями 
git commit - сохраняет все изменения на данный момент 
git diff - вычисляет разницу между git деревьями
git status - показывает какие файлы изменены но не добавлены в индекс и какие ожидают комита
git push - выгружает все изменения в репозиторий
